-> Trabajo Práctico - Programación Funcional en Java

-> Clases principales
 - Alumno.java: representa un alumno con sus atributos y métodos relacionados.
 - Empleado.java: modela un empleado y permite aplicar operaciones sobre sus datos.
 - Libro.java: contiene la estructura y comportamiento de un libro.
 - Producto.java: gestiona la información básica de un producto.
 - Main.java: clase principal que ejecuta distintos ejemplos prácticos utilizando programación funcional.

-> Tecnologías utilizadas
- Lenguaje: Java 17  
- Entorno de desarrollo: IntelliJ IDEA
